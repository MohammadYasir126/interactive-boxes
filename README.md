# Interactive Boxes

This is a simple HTML/CSS and JavaScript project for selecting product units with price, discount, and options like size and color. It mimics a UI typically seen in e-commerce platforms.

## Features

- Selectable product units with radio buttons
- Displays discount and original (strikethrough) prices
- Dropdowns for selecting size and color
- Responsive layout for mobile
- Styled with custom CSS

## Live Preview

[Click here to view](https://mohammadyasir126.github.io/interactive-boxes/)

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
